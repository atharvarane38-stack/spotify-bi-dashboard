## 📌 What This Does
Most music dashboards tell you what happened.
This tool tells you **what to do next.**

Built from the perspective of a BI Analyst at a record label,
this dashboard analyses 600k+ Spotify tracks to identify what
makes songs chart, which artists are about to break out, and
where streaming revenue actually goes.

---

## ✨ Key Features

- 📊 **Executive Overview** — KPIs, genre rankings, release trends
- 🎯 **Hit Predictor** — Audio feature analysis of top vs bottom songs
- 💰 **Revenue Intelligence** — Estimated streaming earnings by artist and genre
- 🌟 **Emerging Artists Radar** — Breakout potential score for rising talent
- 🎧 **Playlist Curator** — Mood-based song recommendations (Hype / Sad / Focus / Party)

## ❓ Business Questions Answered

- Which audio features predict a chart-topping song?
- Which artists show breakout patterns before going mainstream?
- Which genres earn the most per stream — and which are undervalued?
- What mood does a song need to be to maximise first-week streams?
